# SAS Innovate 2026  
## Vector and Complex Number Math using PROC FCMP

This repository contains demo code from my SAS Innovate 2026 session.

---

## Quick Start

Run the programs in this order:

1. 00-data-create.sas  
2. 01-fcmp-build.sas  
3. 02-demo.sas  

---

## What this demonstrates

- Vector math using arrays  
- Custom functions using PROC FCMP  
- Complex number representation in Base SAS  

---

## Requirements

- Base SAS (no SAS/IML required)  
- Tested using SAS Online for Academics
